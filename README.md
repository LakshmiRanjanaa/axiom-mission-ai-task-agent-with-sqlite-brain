# AI Task Agent with SQLite Brain

An intelligent task management agent that uses natural language to create, prioritize, and track tasks.

## Features
- Natural language task creation ("remind me to call mom tomorrow")
- AI-powered task parsing and prioritization
- SQLite database for secure local storage
- Simple Streamlit web interface

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Set your OpenAI API key: `export OPENAI_API_KEY=your_key_here`
3. Run the app: `streamlit run app.py`

## Usage
Just type natural language commands like:
- "Add a high priority task to finish the project report by Friday"
- "Remind me to buy groceries tomorrow"
- "Mark the grocery shopping task as completed"

The AI will understand your intent and manage your tasks accordingly!
